## Summary

ローカル音声アシスタントプロジェクトの基盤となるバックエンド（FastAPI + Python）とフロントエンド（Next.js + TypeScript）の開発環境を構築する。憲法で定められたツールチェーン（uv, pydantic, ruff, mypy, ESLint, Prettier, Jest）を設定し、TDD可能な状態でプロジェクトを初期化する。

**Primary Dependencies**: FastAPI, Pydantic v2, uvicorn (Backend) / Next.js 14+, React 18+ (Frontend)
**Storage**: N/A (初期セットアップのみ)
**Testing**: pytest + pytest-asyncio (Backend) / Jest + React Testing Library (Frontend)
**Target Platform**: Linux/macOS/Windows development environment
**Project Type**: Web application (frontend + backend)
**Performance Goals**: Development環境起動 < 30秒、ヘルスチェックAPI応答 < 100ms
**Constraints**: 憲法準拠（uv必須、ruff必須、TDD必須）
**Scale/Scope**: 開発者1-3名、MVP開発フェーズ
