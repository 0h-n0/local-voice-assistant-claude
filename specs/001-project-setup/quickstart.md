```bash

```

## Backend Setup

```bash
# Navigate to backend directory
cd backend

# Install dependencies with uv
uv sync

# Verify installation
uv run ruff check .
uv run mypy --strict src/

# Run tests
uv run pytest

# Start development server
uv run uvicorn src.main:app --reload --port 8000
```

### Backend Verification

```bash
# Check health endpoint
curl http://localhost:8000/health

# Expected response:
# {"status":"healthy","version":"0.1.0","timestamp":"2025-12-25T10:30:00Z"}
```

## Troubleshooting

### Backend

**Problem**: `uv: command not found`
**Solution**: Install uv following the instructions in Prerequisites section

**Problem**: `ModuleNotFoundError`
**Solution**: Run `uv sync` to install dependencies

**Problem**: Port 8000 already in use
**Solution**: Kill the process using the port or use `--port 8001`

### Frontend

**Problem**: `npm ERR! ENOENT`
**Solution**: Ensure you're in the `frontend` directory and `package.json` exists

**Problem**: Port 3000 already in use
**Solution**: Kill the process or Next.js will automatically use port 3001

### CORS Issues

**Problem**: CORS error when frontend calls backend
**Solution**: Ensure backend is running and CORS middleware is configured for `http://localhost:3000`
